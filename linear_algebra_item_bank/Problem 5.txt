Problem 5
(I)
[Chapter 5, Section 5.2]
A city’s public transport authority models the daily passenger flow between two connected stations, Station A and Station B. Let the state vector

x = [[x_A], [x_B]]

represent the number of passengers at each station at the beginning of the day. The expected passenger movement is modeled by:

A = [[4, 1], [2, 3]]

where the next day’s passenger distribution is given by x_next = Ax.
Find the eigenvalues and eigenvectors of A.
Diagonalize A as A = PDP^-1.
Using the diagonalization, predict the passenger distribution after 3 days if the initial state is x_0 = [[10], [5]].
Also explain the stable movement patterns between the two stations.

Solution:
First of all, we have to find the eigenvalues of the matrix A. To do this, we use the characteristic equation det(A - lambda I) = 0, where lambda represents the eigenvalues and I is the identity matrix.
[Section 5.1 (Introduction to Eigenvalues) -> Section 4.2 (Properties of the Determinant)]

Now, we’ll set up the matrix (A - lambda I):
A - lambda I = [[4, 1], [2, 3]] - [[lambda, 0], [0, lambda]] = [[4-lambda, 1], [2, 3-lambda]]

Next, we calculate the determinant and set it to zero:
(4-lambda)(3-lambda) - (1)(2) = 0
(12 - 7lambda + lambda^2) - 2 = 0
lambda^2 - 7lambda + 10 = 0

Solving the quadratic equation we get:
(lambda - 5)(lambda - 2) = 0

Hence, our eigenvalues are lambda_1 = 5 and lambda_2 = 2.

Now, we have to find the eigenvectors corresponding to each eigenvalue by solving the nullspace equation (A - lambda I)x = 0.
[Section 5.1 (Introduction to Eigenvalues) -> Section 2.2 (Solving for Ax = 0 and Ax = b)]

For lambda_1 = 5:
[[4-5, 1], [2, 3-5]] [[x_A], [x_B]] = [[0], [0]]
[[-1, 1], [2, -2]] [[x_A], [x_B]] = [[0], [0]]

This gives us the equation -x_A + x_B = 0 => x_A = x_B.
Let x_A = 1, then x_B = 1.
Our first eigenvector is v_1 = [[1], [1]].

For lambda_2 = 2:
[[4-2, 1], [2, 3-2]] [[x_A], [x_B]] = [[0], [0]]
[[2, 1], [2, 1]] [[x_A], [x_B]] = [[0], [0]]

This gives us the equation 2x_A + x_B = 0 => x_B = -2x_A.
Let x_A = 1, then x_B = -2.
Our second eigenvector is v_2 = [[1], [-2]].

In order to diagonalize the matrix as A = PDP^-1, we construct matrix P using our eigenvectors as columns, and matrix D as a diagonal matrix containing our eigenvalues.
[Section 5.2 (Diagonalization of a Matrix) -> Section 1.6 (Inverses and Transposes)]

P = [[1, 1], [1, -2]], D = [[5, 0], [0, 2]]

We also need the inverse of P (P^-1). Using the 2x2 inverse formula 1/det(P) [[d, -b], [-c, a]]:
det(P) = (1)(-2) - (1)(1) = -3
P^-1 = -1/3 [[-2, -1], [-1, 1]] = [[2/3, 1/3], [1/3, -1/3]]

Now, we use the diagonalization to predict the passenger distribution after 3 days. The formula for the state after k days is x_k = P D^k P^-1 x_0.
[Section 5.3 (Difference Equations and Powers A^k)]

For k = 3 and x_0 = [[10], [5]]:
x_3 = [[1, 1], [1, -2]] [[5^3, 0], [0, 2^3]] [[2/3, 1/3], [1/3, -1/3]] [[10], [5]]

First, multiply P^-1 x_0:
[[2/3, 1/3], [1/3, -1/3]] [[10], [5]] = [[20/3 + 5/3], [10/3 - 5/3]] = [[25/3], [5/3]]

Next, multiply by D^3 = [[125, 0], [0, 8]]:
[[125, 0], [0, 8]] [[25/3], [5/3]] = [[3125/3], [40/3]]

Finally, multiply by P:
x_3 = [[1, 1], [1, -2]] [[3125/3], [40/3]] = [[3125/3 + 40/3], [3125/3 - 80/3]] = [[3165/3], [3045/3]] = [[1055], [1015]]

After 3 days, there will be 1,055 passengers at Station A and 1,015 at Station B.

The dominant eigenvalue lambda_1 = 5 corresponds to the eigenvector v_1 = [[1], [1]]. This indicates that over time, the dominant long-term stable pattern is a 1:1 distribution of passengers between the two stations, multiplying by a factor of 5 daily. The secondary pattern v_2 = [[1], [-2]] with lambda_2 = 2 grows much more slowly and its influence quickly fades compared to the dominant 1:1 trend.

(II)
[Chapter 5, Section 5.2]
An ecologist models the yearly population of two interdependent bird species, Species A and Species B, in a wildlife reserve. Let the state vector

x = [[x_A], [x_B]]

represent the population of each species (in thousands) at the beginning of the year. The expected population growth and interaction is modeled by:

A = [[3, 2], [1, 2]]

where the next year's population distribution is given by x_next = Ax.

Find the eigenvalues and eigenvectors of A.
Diagonalize A as A = PDP^-1.
Using the diagonalization, predict the population distribution after 3 years if the initial state is x_0 = [[5], [4]].
Also explain the stable growth patterns between the two species.

Solution:
First of all, we have to find the eigenvalues of the matrix A. To do this, we use the characteristic equation det(A - lambda I) = 0, where lambda represents the eigenvalues and I is the identity matrix.
[Section 5.1 (Introduction to Eigenvalues) -> Section 4.2 (Properties of the Determinant)]

Now, we’ll set up the matrix (A - lambda I):
A - lambda I = [[3, 2], [1, 2]] - [[lambda, 0], [0, lambda]] = [[3-lambda, 2], [1, 2-lambda]]

Next, we calculate the determinant and set it to zero:
(3-lambda)(2-lambda) - (2)(1) = 0
(6 - 5lambda + lambda^2) - 2 = 0
lambda^2 - 5lambda + 4 = 0

Solving the quadratic equation we get:
(lambda - 4)(lambda - 1) = 0

Hence, our eigenvalues are lambda_1 = 4 and lambda_2 = 1.

Now, we have to find the eigenvectors corresponding to each eigenvalue by solving the nullspace equation (A - lambda I)x = 0.
[Section 5.1 (Introduction to Eigenvalues) -> Section 2.2 (Solving for Ax = 0 and Ax = b)]

For lambda_1 = 4:
[[3-4, 2], [1, 2-4]] [[x_A], [x_B]] = [[0], [0]]
[[-1, 2], [1, -2]] [[x_A], [x_B]] = [[0], [0]]

This gives us the equation -x_A + 2x_B = 0 => x_A = 2x_B.
Let x_B = 1, then x_A = 2.
Our first eigenvector is v_1 = [[2], [1]].

For lambda_2 = 1:
[[3-1, 2], [1, 2-1]] [[x_A], [x_B]] = [[0], [0]]
[[2, 2], [1, 1]] [[x_A], [x_B]] = [[0], [0]]

This gives us the equation x_A + x_B = 0 => x_A = -x_B.
Let x_B = -1, then x_A = 1.
Our second eigenvector is v_2 = [[1], [-1]].

In order to diagonalize the matrix as A = PDP^-1, we construct matrix P using our eigenvectors as columns, and matrix D as a diagonal matrix containing our eigenvalues.
[Section 5.2 (Diagonalization of a Matrix) -> Section 1.6 (Inverses and Transposes)]

P = [[2, 1], [1, -1]], D = [[4, 0], [0, 1]]

We also need the inverse of P (P^-1). Using the 2x2 inverse formula 1/det(P) [[d, -b], [-c, a]]:
det(P) = (2)(-1) - (1)(1) = -3
P^-1 = -1/3 [[-1, -1], [-1, 2]] = [[1/3, 1/3], [1/3, -2/3]]

Now, we use the diagonalization to predict the population distribution after 3 years. The formula for the state after k years is x_k = P D^k P^-1 x_0.
[Section 5.3 (Difference Equations and Powers A^k)]

For k = 3 and x_0 = [[5], [4]]:
x_3 = [[2, 1], [1, -1]] [[4^3, 0], [0, 1^3]] [[1/3, 1/3], [1/3, -2/3]] [[5], [4]]

First, multiply P^-1 x_0:
[[1/3, 1/3], [1/3, -2/3]] [[5], [4]] = [[5/3 + 4/3], [5/3 - 8/3]] = [[3], [-1]]

Next, multiply by D^3 = [[64, 0], [0, 1]]:
[[64, 0], [0, 1]] [[3], [-1]] = [[192], [-1]]

Finally, multiply by P:
x_3 = [[2, 1], [1, -1]] [[192], [-1]] = [[384 - 1], [192 + 1]] = [[383], [193]]

After 3 years, the population will be 383,000 for Species A and 193,000 for Species B.

Now we can interpret the patterns. The dominant eigenvalue lambda_1 = 4 corresponds to the eigenvector v_1 = [[2], [1]]. This indicates that the long-term stable ecological pattern tends strictly toward a 2:1 ratio of Species A to Species B, with the total joint population quadrupling every year. The secondary pattern v_2 = [[1], [-1]] with lambda_2 = 1 is stagnant (it neither grows nor shrinks), meaning initial imbalances strictly wash out over time in favor of the dominant 2:1 growth trend.
