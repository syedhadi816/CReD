Problem 10
(I)
[Chapter 7, Section 7.2]
An engineer is designing spring systems for three different machines. The stiffness of each system can be represented by a 2 x 2 positive definite matrix K, which models how the system resists deformation. The matrices are:

K_1 = [[100, 0], [0, 2]], K_2 = [[2, 1], [1, 2]], K_3 = [[3, 1], [1, 1]].

For each matrix, compute the largest eigenvalue lambda_max and the condition number kappa = lambda_max / lambda_min. Also tell which spring system is more sensitive to applied forces based on its condition number.

Solution:
First of all, we need to find the eigenvalues for each matrix by solving the characteristic equation det(K - lambda I) = 0.
[Section 7.2 (Matrix Norm and Condition Number) -> Section 5.1 (Introduction to Eigenvalues)]

For the first matrix K_1:
Because K_1 is a diagonal matrix, its eigenvalues are simply the entries on its main diagonal.
lambda_1 = 100, lambda_2 = 2

The largest eigenvalue is lambda_max = 100, and the smallest is lambda_min = 2.
The condition number is:
kappa_1 = lambda_max / lambda_min = 100 / 2 = 50

For the second matrix K_2:
We solve the characteristic equation:
det [[2-lambda, 1], [1, 2-lambda]] = (2-lambda)^2 - 1 = 0
lambda^2 - 4lambda + 4 - 1 = 0 => lambda^2 - 4lambda + 3 = 0

Factoring the quadratic equation:
(lambda - 3)(lambda - 1) = 0 => lambda_1 = 3, lambda_2 = 1

The largest eigenvalue is lambda_max = 3, and the smallest is lambda_min = 1.
The condition number is:
kappa_2 = lambda_max / lambda_min = 3 / 1 = 3

For the third matrix K_3:
We solve the characteristic equation:
det [[3-lambda, 1], [1, 1-lambda]] = (3-lambda)(1-lambda) - 1 = 0
lambda^2 - 4lambda + 3 - 1 = 0 => lambda^2 - 4lambda + 2 = 0

Using the quadratic formula:
lambda = (4 +/- sqrt(16 - 8)) / 2 = (4 +/- sqrt(8)) / 2 = 2 +/- sqrt(2)

The largest eigenvalue is lambda_max = 2 + sqrt(2) approx 3.414, and the smallest is lambda_min = 2 - sqrt(2) approx 0.586.
The condition number is:
kappa_3 = (2 + sqrt(2)) / (2 - sqrt(2))

To simplify, we multiply the numerator and denominator by the conjugate (2 + sqrt(2)):
kappa_3 = (2 + sqrt(2))^2 / ((2 - sqrt(2))(2 + sqrt(2))) = (4 + 4sqrt(2) + 2) / (4 - 2) = (6 + 4sqrt(2)) / 2 = 3 + 2sqrt(2) approx 5.828

Comparing the condition numbers, we have kappa_1 = 50, kappa_2 = 3, and kappa_3 approx 5.828.
The first spring system (K_1) has by far the highest condition number (kappa_1 = 50). Therefore, it is the most highly sensitive to applied forces, meaning small changes or errors in the load can cause drastically skewed and highly unequal deformations along its principal axes.

(II)
[Chapter 7, Section 7.2]
A civil engineer is analyzing the structural stiffness of three different building foundations. The stiffness of each foundation can be represented by a 2 x 2 positive definite matrix M, which models how the foundation resists settlement. The matrices are:

M_1 = [[50, 0], [0, 5]], M_2 = [[4, 2], [2, 4]], M_3 = [[5, 2], [2, 2]].

For each matrix, compute the largest eigenvalue lambda_max and the condition number kappa = lambda_max / lambda_min. Also tell which foundation system is more sensitive to uneven loads based on its condition number.

Solution:
First of all, we need to find the eigenvalues for each matrix by solving the characteristic equation det(M - lambda I) = 0.
[Section 7.2 (Matrix Norm and Condition Number) -> Section 5.1 (Introduction to Eigenvalues)]

For the first matrix M_1:
Because M_1 is a diagonal matrix, its eigenvalues are simply the entries on its main diagonal.
lambda_1 = 50, lambda_2 = 5

The largest eigenvalue is lambda_max = 50, and the smallest is lambda_min = 5.
The condition number is:
kappa_1 = lambda_max / lambda_min = 50 / 5 = 10

For the second matrix M_2:
We solve the characteristic equation:
det [[4-lambda, 2], [2, 4-lambda]] = (4-lambda)^2 - 4 = 0
lambda^2 - 8lambda + 16 - 4 = 0 => lambda^2 - 8lambda + 12 = 0

Factoring the quadratic equation:
(lambda - 6)(lambda - 2) = 0 => lambda_1 = 6, lambda_2 = 2

The largest eigenvalue is lambda_max = 6, and the smallest is lambda_min = 2.
The condition number is:
kappa_2 = lambda_max / lambda_min = 6 / 2 = 3

For the third matrix M_3:
We solve the characteristic equation:
det [[5-lambda, 2], [2, 2-lambda]] = (5-lambda)(2-lambda) - 4 = 0
lambda^2 - 7lambda + 10 - 4 = 0 => lambda^2 - 7lambda + 6 = 0

Factoring the quadratic equation:
(lambda - 6)(lambda - 1) = 0 => lambda_1 = 6, lambda_2 = 1

The largest eigenvalue is lambda_max = 6, and the smallest is lambda_min = 1.
The condition number is:
kappa_3 = lambda_max / lambda_min = 6 / 1 = 6

Comparing the condition numbers, we have kappa_1 = 10, kappa_2 = 3, and kappa_3 = 6.
The first foundation system (M_1) has the highest condition number (kappa_1 = 10). Therefore, it is the most sensitive to uneven loads. In engineering terms, this ill-conditioning indicates that the foundation is much stiffer in one direction than the other, meaning slightly asymmetric loading could cause severe, disproportionate differential settlement.
